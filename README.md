# nekomimi-bot </br>
## The cutest discord bot! </br>

- Plays music
- Super kawaii
- Customizable

### Music Commands
"!play (!p) {youtube search}" </br>
Searches a song, then adds it to the queue </br>
Example: !p rick astley </br>

"!queue (!q)" </br>
Displays the current queue </br>
Example: !queue </br>

### Terminal Commands
">{command}" </br>
Runs a command on the host </br>
Example: >ipconfig </br>
